# Exemplo SignUp

## Instalação


Primeiramente clone o projeto deste repositório

```bash
  git clone https://github.com/Pedro-Alcantara-M/signup-example.git
```

Instale projeto com yarn

```bash
  yarn install
```

Inicie o projeto 

```bash
  yarn start
```

E abra o navegadoor com o link gerado pelo vite.
